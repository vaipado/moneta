# 🪙 Moneta
> Gestão financeira pessoal simplificada para controle de entradas e saídas.

O **Moneta** é uma aplicação web desenvolvida para auxiliar usuários a organizarem suas finanças diárias de forma intuitiva, permitindo a visualização clara de saldos, receitas e despesas.

---

## 🚀 Tecnologias
O projeto foi construído utilizando o ecossistema moderno do React:

* **React** — Biblioteca core para interface.
* **Vite** — Ferramenta de build de alta performance.
* **CSS Modules** — Estilização isolada para evitar conflitos de escopo.
* **ESLint** — Garantia de qualidade e padronização do código.

---

## 📋 Funcionalidades
Acompanhe o progresso do desenvolvimento:

- [x] **Setup Inicial**: Estrutura base e limpeza de boilerplate.
- [x] **Dashboard**: Header responsivo e cards com resumo de saldos (Entradas, Saídas, Total).
- [x] **Feed**: Listagem dinâmica de transações com filtros por tipo.
- [x] **Formulário**: Cadastro de novas transações via Modal acessível.
- [x] **Persistência**: Salvamento automático de dados no `LocalStorage` do navegador.
